Glossary

Associate

A modeled companion to a Tk widget or widget-like resource.

Castle

A runtime participant that owns state, inbox, outbox, associates, child castles, spots, placements, and optional handlers/reconcilers.

Desired

The projection target: values the castle or RT wants projected onto the concrete widget.

Observed

The associate's public modeled-observation surface. It is intentionally maintained, but not a complete mirror of Tk.

Private

Associate implementation detail such as trace IDs, private variables, projected-value caches, or suppression flags.

Route

A runtime record that moves messages from one outbox to another inbox.

Spot

A named place in a castle's layout tree.

Template

A Python callable that returns a castle specification. Templates describe what should be built; they are not live castles.
