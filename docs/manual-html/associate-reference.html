Associate Reference

Each associate type is documented by the Little Castles promise it makes, not just by the Tk widget underneath.

Reference Page Format

  • Name
  • Purpose
  • Underlying Tk widget
  • Can host children?
  • Embeddable?
  • Desired fields
  • Observed fields
  • Private resources
  • Default events
  • Optional events
  • Projection behavior
  • Destroy behavior

Current Associates

Name Purpose Defaults Observed
window Owns a Toplevel and content frame. window_resized actual_width, actual_height
button Command-style button. button_pressed None currently modeled.
label Displays text. None. None currently modeled.
entry Single-line text input. submitted text, focused

Entry Notes

The Entry associate uses a private StringVar and trace to keep modeled text observation current. Projection suppresses text_changed emission.
